## Description
An OpenGL project where three rooms are created with geometric data from a txt file, the camera movement is constrained to the withing the rooms so it can move outside. In between the rooms there are conflict free portals that allow viewing and traversal between the rooms. There are also smaller objects placed in each room that also has collision detection with the camera.
